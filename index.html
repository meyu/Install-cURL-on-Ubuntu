所為何來

於 Ubuntu 上安裝 cURL 工具，以便使用 curl 指令來傳送檔案。

使用環境

Ubuntu Server 12.04

安裝方式

於終端機中，輸入以下指令即可：

sudo aptitude install curl



補充說明

cURL

是一個命令列工具，可使用 HTTP、FTP 等網路協定來傳送檔案。
與 wget 類似的是，都可用來下載檔案；不同的是，cURL 支援帳密登入，還能上傳檔案，並可使用萬用字元來完成批次傳輸。

簡要
=
於 Ubuntu 上安裝 cURL 工具，以便使用 url 指令來傳輸檔案。

使用環境
=
Ubuntu Server 12.04

安裝方式
=
於終端機中，輸入以下指令即可：
```bash
sudo aptitude install curl
```

<br>
<br>

補充說明
=
###cURL
>是一個命令列工具，用於傳輸符合 URL 語法的檔案。  
>與 wget 類似的是，都可用來下載檔案；不同的是，cURL 還能上傳檔案，並可使用萬用字元來完成批次傳輸。
